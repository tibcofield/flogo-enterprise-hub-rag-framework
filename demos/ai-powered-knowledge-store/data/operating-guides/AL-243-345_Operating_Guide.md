## 6. Operating Instructions

### Starting the Engine (Electric Start)
1. Place the mower on level ground.
2. Set the throttle control to **START / FAST**.
3. Ensure the battery is connected.
4. Press and hold the **START** button until the engine runs.
5. Release the button immediately after starting.

### Starting the Engine (Manual Recoil)
If the battery is discharged:
1. Set throttle to **START / FAST**.
2. Pull the recoil starter firmly.

### Mowing
- Operate at a steady walking pace.
- For tall or dense grass, increase cutting height and mow progressively.
- Do not force the mower through heavy growth.

### Stopping the Engine
- Move the throttle control to **STOP**.
- Wait until the blade has completely stopped before handling.

## 7. Cutting Height Adjustment

The central height adjustment system allows cutting heights from **25 mm to 80 mm**.

1. Stop the engine and disconnect the spark plug lead.
2. Pull the height adjustment lever outward.
3. Select the desired height.
4. Release the lever to lock.
